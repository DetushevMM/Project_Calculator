# CMake Build System

Данная папка содержит систему сборки проекта на основе CMake и набор модульных тестов.

## Содержимое

CMake/
├── CMakeLists.txt
├── README.md
├── tokenize_test.cpp
├── parser_test.cpp
└── calculator_test.cpp
## Назначение файлов

### CMakeLists.txt

Основной файл конфигурации CMake.

Отвечает за:

- настройку проекта;
- подключение стандарта C++17;
- подключение исходных файлов;
- подключение библиотеки GoogleTest;
- сборку тестов;
- регистрацию тестов в CTest.

---

### tokenize_test.cpp

Модульные тесты токенизатора.

Проверяются:

- числа;
- операторы;
- скобки;
- функции;
- неизвестные токены;
- корректность формирования последовательности Token.

---

### parser_test.cpp

Модульные тесты синтаксического анализатора.

Проверяются:

- приоритет операций;
- работа со скобками;
- построение дерева выражения;
- обработка ошибок разбора.

---

### calculator_test.cpp

Интеграционные тесты калькулятора.

Проверяется полный цикл:

Строка
    ↓
Tokenize
    ↓
Parser
    ↓
AST
    ↓
Calculate()
Тестируются:

- арифметические выражения;
- вложенные скобки;
- унарные операции;
- ошибки вычислений;
- исключения.

---

# Требования

Для сборки необходимы:

- CMake 3.16 или выше;
- компилятор с поддержкой C++17;
- Git (для автоматической загрузки GoogleTest).

---

# Сборка проекта

Создать каталог сборки:

mkdir build
cd build
Сгенерировать проект:

cmake ../CMake
Собрать:

cmake --build .
---

# Запуск тестов

Запустить все тесты:

ctest
или

ctest --output-on-failure
---
